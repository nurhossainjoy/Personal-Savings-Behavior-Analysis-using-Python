# Personal Saving Behavior Analysis

This project investigates the determinants of Personal Saving Behavior (PSB) using survey data and statistical modeling techniques in Python.

## Research Objective
The objective of this project is to examine how financial literacy, income constraints, social influence, financial access, and risk preference influence personal saving behavior.

## Dataset
The dataset contains survey responses from 250 individuals with 39 variables measuring different constructs.

## Methodology

The following statistical methods were applied:

• Cronbach's Alpha (Reliability Test)
• Composite Reliability
• Average Variance Extracted (AVE)
• Fornell–Larcker Criterion
• Correlation Analysis
• Multiple Linear Regression (OLS)

## Tools & Libraries

Python libraries used:

- Pandas
- NumPy
- Seaborn
- Matplotlib
- Statsmodels
- Scikit-learn

## Key Findings

• Financial Literacy significantly influences saving behavior  
• Income Constraint also has a significant effect  
• Social Influence shows a negative significant impact  
• Financial Access and Risk Preference were not statistically significant
